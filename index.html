Reriid

Final Degree Proyect

Reriid is a project whose aim is to connect people nearby to exchange/sell books. It uses geolocation to display the closest posts to the user.

Developed using Ionic and Laravel with Websockets, it incorporates a realtime chat functionality. The deployment for this project was made using AWS ElasticBeanstalk.
